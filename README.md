# `lsurl`
Print paths (files or directories) as `file://` URLs.

Usage:
```bash
$ lsurl
file:///Users/kms/
$ lsurl .
file:///Users/kms/
$ lsurl /Users
file:///Users/
$ lsurl /Users/*
file:///Users/Guest/
file:///Users/Shared/
file:///Users/kms/
```
